# Documentation

This directory contains project documentation and presentation materials.

## Structure

```
docs/
├── api/              # API documentation
├── architecture/     # Architecture diagrams
├── deployment/       # Deployment guides
├── presentation/     # Exam presentation materials
└── README.md
```

## Contents

- API Documentation: Endpoints, request/response formats
- Architecture: System design, component diagrams
- Deployment: Setup guides, Docker instructions
- Presentation: Exam materials, slides, demos

## Adding Documentation

1. Create appropriate subdirectory
2. Add markdown files with clear naming
3. Update this README with links
